Project Description

This project is a full-stack web application that provides URL shortening for users and provides a few statisticss. The backend is built with Go using the Gin framework and MySQL for data storage. After authentification, users can create shortened URLs that redirect to long destination links. For each short URL, the system tracks basic statistics such as total clicks and location.

The frontend is built with React (Vite) and offers a simple interface where users can manage their links, customize them and view the statistics for each one.
